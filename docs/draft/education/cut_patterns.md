# 頻出パターン（教育）

Status: Draft
Summary: 中学受験算数の立体切断で頻出する「型」を、教え方（見るべき線/強調/説明方針）として整理する。

## 目的
- 学習者が「型で見る」ための軸を提供し、思考の迷走を減らす
- UI の強調表示と解説文が、学習の狙いに沿うように基準を固定する

## パターン分類（教育用）
### 例: 3頂点切断
- ねらい: 空間対角線や対応辺を意識させる
- まず見る: 空間対角線、切断面の三角形の辺
- よくあるつまずき: 立体の「奥」を見落とす（平面図的に捉える）

### 例: 辺3点・同じ比
- ねらい: 相似/比例の見方を固定する
- まず見る: 垂直な3辺、比が等しいこと
- よくあるつまずき: 比の対応（どの辺のどちら側か）が崩れる

## ハイライト方針（教育）
- 最初は情報量を増やしすぎない（「必ず見る線」だけを強調する）
- 強調は「理由とセット」で提示する（なぜその線を見るのか）
- 色や太さの具体値は UI 実装で調整し、ここでは“何を強調するか”を正とする

## 解説文方針（教育）
- 断定よりも「見る順序」を提示する（手順の可視化）
- SnapPointID 等の内部表現を学習者に露出させない（必要ならラベルで置換）
