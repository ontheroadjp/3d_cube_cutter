# 172 face color theme

- 実装(技術)について
  - DisplayState に faceColorTheme を追加し、UIManager/React サイドパネルからテーマを切替可能にした。
  - Cube 側でテーマ別パレットと向かい面グルーピングを行い、同一色を保証しつつ既存のハッシュ色選択も維持した。
  - setFaceColorTheme でキャッシュをクリアし、既存メッシュの色を即時更新する。

- 仕様(L1ドキュメント)について
  - 表示設定は Presentation/DisplayState として保持し、保存/復元が可能な構成に合わせた。

- 数学的観点から
  - 面の向き（前後/上下/左右）は FaceID の構造定義に紐づくため、向かい面の同色化は構造の整合性を崩さない。

- ユーザー体験(ユーザー価値)について
  - 学習者が視認しやすい配色テーマを選択でき、向かい面が同色で認識しやすくなる。

- 追加対応
  - 左右ペアと前後ペアの色割当を入れ替え、デフォルトテーマを青系に変更。
- 追加対応
  - 光源の影響を受けないよう面材質を MeshBasicMaterial に変更し、常に同じ色で表示。
  - 展開開始時に底面ハイライトを解除して元の色へ戻す。
