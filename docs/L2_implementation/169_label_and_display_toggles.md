# 169 label and display toggles

- 実装(技術)について
  - 頂点ラベルは buildCubeStructure に fallbackLabelMap を渡し、A-H の既定ラベルを付与するようにした。
  - 面ラベルは FaceID から前/後/上/下/左/右をマッピングし、切断面は「切断面」として表示。
  - 面の色は元の FaceID を基準に固定し、切断後も色が変わらないようにした。
  - 切断面表示のトグルは SSOT 側の面メッシュに適用するため、Cube に cutFaceVisible を持たせて切断面だけを非表示にする。
  - 辺ラベルのポップアップは SelectionManager 内で EdgeID ベースに統一し、切断後でもポップアップ表示できるようにした。
  - 切断点の頂点ラベルは isCutPoint を参照して抑制し、ID が出ないようにした。
  - 切断点マーカーに userData を付与し、hover 時の分割表示を強制できるようにした。

- 仕様(L1ドキュメント)について
  - 表示トグルは UI の DisplayState と同期して SSOT 表示にも反映されるべきため、Cut 表示の適用経路に Cube 側の切断面表示制御を追加した。

- 数学的観点から
  - 面ラベルの割当は NetManager の faceId 定義と一致する FaceID に基づいて行い、向きや順序のズレが起きないようにした。

- ユーザー体験(ユーザー価値)について
  - 辺の長さポップアップが hover で復帰し、必要なときにだけ確認できるようになった。
  - 前後左右の面名が直感的に表示されるため、切断面の理解が容易になる。
